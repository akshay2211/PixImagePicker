# Overview

## What is Pix?

Pix is a WhatsApp-style image and video picker library for Android. It provides a modern, intuitive user interface for selecting and capturing media files in your application.

## Why Choose Pix?

- **Production Ready** - Used by thousands of developers and included in Google's Dev Library
- **Beautiful UI** - Designed following Android Material Design principles
- **Lightweight** - Minimal APK size impact (~500KB)
- **Feature Rich** - Supports images, videos, camera capture, and more
- **Well Maintained** - Regular updates and active community support
- **Easy Integration** - Simple API with comprehensive documentation

## Key Capabilities

### Media Selection
- Pick multiple images and/or videos
- Camera capture (photo and video)
- Customizable grid layout
- Fast scroll support

### Configuration
- Set image/video count limits
- Choose capture ratio (4:3, 16:9, auto)
- Specify storage path
- Pre-select items
- Configure flash settings

### Customization
- Theme colors to match your app
- Custom callbacks and event handling
- Integration with Navigation, ViewPager2, Fragments
- Multiple integration patterns

## Platform Support

- **Minimum API Level**: 16 (Android 4.1+)
- **Target API Level**: Latest (currently 34+)
- **Build System**: Gradle with Kotlin DSL
